# 🛡️ Week 4 - SOC Operations

## 📌 Overview
This project demonstrates a complete SOC workflow including detection, triage, response, automation, and reporting.

## 🧰 Tools Used
- Wazuh
- TheHive
- CrowdSec
- Metasploit
- MITRE Caldera
- Elastic Security
- Velociraptor

## ⚔️ Attack Simulation
A Samba vulnerability was exploited using Metasploit (T1210).

## 🚨 Detection
Wazuh detected suspicious activity and generated alerts.

## 🔍 Triage
Alerts were analyzed and investigated using TheHive.

## ⚡ Response
- Attacker IP blocked using CrowdSec
- Infected system isolated

## 🤖 Automation
SOAR playbook created for automatic response.

## 📊 Metrics
- MTTD: 2 hours
- MTTR: 4 hours
- Dwell Time: 6 hours
